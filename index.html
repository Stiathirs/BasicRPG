Welcome to BasicRPG!

BasicRPG is an RPG-style grinding game where you beat monsters to become stronger!

Do you get bored from grinding? Don't worry, this game makes it even more boring by having no graphics! But at least there's some different mechanics to make up for it I guess :P

  • Kill monsters and level up to become stronger!
  • Get gold and become rich!
  • Die a horrible death I guess?
  • Get weapons and armor!
  • Die again?

Join the dedicated Discord Server for the game!

This game is in development, after an update, your save data may be wiped, YOU HAVE BEEN WARNED

This game is open-source, and you can view the repository here!

To import the file into Twine, drag the index.html from either Play/index.html or Legacy/index.html into the Twine Editor.

You can play the modern version here!

Want to play the prerelease version? The prerelease version of the game is here, but it will no longer be updated. Your progress here also will not be saved.

About the Developer

Oh, hello there! I am Stiathirs the Protogen, the sole developer of BasicRPG. This game has been a bit of a passion project of mine which started in the summer of 2018.

Development has been on and off throughout the years, and I eventually stopped working on the project in the fall of 2019. The project originally started because of something I had saw in the Twine Cookbook, which was RPG mechanics. This inspired me to make a game which envolved RPG-like mechanics, and to push Twine (and Harlowe) to it's limits. Now that I've come back in 2022, I've seen just how terrible my game was, and wanted to improve it.

Over time, I wasted away developing this game that I thought nobody would even want to play. However, this didn't seem to be completely true, as I actually bothered to write over 100 more kilobytes of text improving the game and expanding upon what was originally just a concept which would never come to completion. If the past me looked at this without any knowledge of why I continued working on this abandoned project, they'd laugh at the fact I bothered to expand it.

Over the course of developing this game, I've learned about just how much potential the Harlowe engine (and Twine in general) really has. It's not just a gamebook maker, it's an entire game development engine. Granted, it might not be your traditional game development engine like Unity or Unreal, it's still very powerful and allows you to make a game that nobody else would bother doing in either.

Funny thing is, I used to want to be a gamedev. Around the time I abandoned this project back in 2019 was when I realized I didn't want to be a professional gamedev. Now that I've come back to this game and started developing it again, I realize I want game making to be a hobby, not a career. I want to enjoy what I'm making, and not have someone boss me around to finish a game that doesn't interest me.

Overall, this has been (and still is) a fun project to work on, and I hope that you can see the passion I've put into this game. Remember to revisit projects you abandon, maybe you'll re-ignite your passion for it.

    - Stiathirs the Protogen
